# AppoApps

WHAT IS AppoApps?
- Using AppoApps, you can launch all your apps from one place.

Intelligent app search:
- In AppoApps, you can select the folders in which you want to search for apps.
- You can also enter a name to search for an app with that name in the selected folders!
(DEFAULT FOLDERS: C:\Program Files, C:\Program Files (x86), C:\ProgramData, C:\Users\<username>\AppData\Roaming)

- PROGRAMMING LANGUAGES: JAVA
