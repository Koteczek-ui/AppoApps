# AppoApps

**WHAT IS AppoApps?**
  - Using AppoApps, you can launch all your apps from one place.

**Smart App Search:**
  - In AppoApps, you can select the folders in which you want to search for apps.
  - You can also enter a name to search for an app with that name in the selected folders!
  (DEFAULT FOLDERS: C:\Program Files, C:\Program Files (x86), C:\ProgramData, C:\Users\<username>\AppData\Roaming)

**Features:**
  - Easy app launch
  - Themes
  - Easy app search
  - Settings file saved to C:\Users\<username>\AppData\Roaming\AppoApps\settings.json
  - Last search and selected folders saved to C:\Users\<username>\AppData\Roaming\AppoApps\session.json
  - Automatically loads sessions and settings when AppoApps starts
  - Automatically saves sessions and settings when AppoApps closes

**Requirements:**
  - Java

- *PROGRAMMING LANGUAGES: JAVA*
- *VERSION DESIGNATION: yyyymmddbb (y=year, m=month, d=day, b=build)*
